# Multi Chain Wallet Platform

This project is a multi-chain wallet platform that allows users to manage their cryptocurrency assets across multiple blockchains. The platform provides a user-friendly interface for managing wallets, viewing balances, and performing transactions.

```
apps ├── frontnend
     ├── src
     │   ├── components
     │   ├── pages
     │   ├── services
     │   └── utils
     ├── public
     └── package.json
     |--api-gateway
      |--src
          |--controllers
          |--routes
          |--services
services
  |--tx-service
      |--src
          |--controllers
          |--services
  |--crypto-service
      |--src
          |--controllers
          |--services
```
